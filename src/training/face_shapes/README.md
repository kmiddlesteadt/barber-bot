# Face Shape Training Images

This folder is organized for `torchvision.datasets.ImageFolder`:

- `Oval/`
- `Square/`
- `Round/`
- `Heart/`
- `Oblong/`

The current JPG files are synthetic placeholder illustrations. They are useful
for confirming that the folder structure and training pipeline work, but they
are not enough to train a reliable real-world face-shape classifier.

For better model accuracy, replace or expand these placeholders with many
front-facing real face photos for each class.
